About HTML Elements

Headings

HTML provides six levels of headings, from h1 (most important) to h6 (least important). This page uses h1, h2, and h3 tags to create a clear hierarchy.

Paragraphs

The paragraph tag (<p>) is used to define blocks of text. Paragraphs automatically add spacing before and after the content.

Links

Links are created using the anchor tag (<a>). You can link to other pages, sections on the same page, or external websites. Try clicking the navigation menu above to jump to different sections!
